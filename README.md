House Price Prediction System
Overview
This project predicts house prices using Machine Learning based on features like location, size, and other parameters. It includes data analysis, visualization, and an interactive Streamlit web application.

Features
Data preprocessing and cleaning

Exploratory Data Analysis (EDA)

Machine Learning model training

Price prediction system

Interactive Streamlit web app

Tech Stack
Python

Pandas, NumPy

Matplotlib, Seaborn

Scikit-learn

Streamlit, Plotly

Dependencies managed using requirements.txt.

Installation
pip install -r requirements.txt
Run the Project
Run Python Script
python house_price_prediction.py
Run Streamlit App
streamlit run app.py
Output
Predicts house prices based on input features

Displays graphs and visual insights

Interactive UI for user input and prediction
